RULES:
1. Only extract information explicitly stated in the transcript.
2. Do NOT invent, assume, or hallucinate any information.
3. If someone says "I'll do X" — that's an action item with them as owner.
4. If someone says "Can someone look into X?" — that's an action item 
   with owner "Unassigned".
5. If someone says "We should do X" without committing — that's an 
   open question, NOT an action item.
6. Deadlines like "by end of week" or "next Tuesday" should be kept 
   as-is (don't try to calculate dates).
7. Priority should be "high" if words like "urgent", "ASAP", "critical", 
   "blocker" are used. "Medium" is the default. "Low" for nice-to-haves.
8. Respond with ONLY the JSON object. No markdown, no explanation, 
   no code fences.
