# Sample Meeting Transcripts

Sample audio files and transcripts are excluded from this repo 
for privacy. To test the system yourself:

1. Record a 2-minute fake meeting on your phone with a freind. Make sure to include:
   - A decision
   - Action items
   - An open question

2. Save as .mp3 or .wav in this folder

3. Run `python src/transcribe.py --input samples/your_file.mp3`

A demo output from an anonymized transcript is available at 
`samples/demo_output.json`.
